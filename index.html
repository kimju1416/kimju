앱 설치 방법
kimju.zip의 앱은 Google Apps Script로 만들어진 웹앱이에요.
아래 순서대로 따라하시면 누구나 5분 안에 설치하실 수 있어요!
1
공유 링크에서 내 드라이브로 사본 만들기
위 목록에서 원하는 앱 링크를 클릭하면 구글 드라이브 사본 만들기 화면이 열려요.
'사본 만들기' 버튼을 눌러 선생님 계정의 드라이브로 가져오세요.
⚠ 이 단계가 가장 중요해요! 사본이 있어야 직접 수정·배포 권한이 생깁니다.
2
Apps Script 에디터 열기
사본 스프레드시트를 열고 상단 메뉴에서
'확장 프로그램' → 'Apps Script' 를 클릭해 스크립트 에디터를 여세요.
💡 '스크립트 편집기'로 표시되는 경우도 있어요. 같은 기능이니 걱정 마세요!
3
새 배포 시작하기
에디터 오른쪽 위 '배포' 버튼(✈️)을 클릭하고
드롭다운에서 '새 배포'를 선택하세요.
4
배포 설정하기
'새 배포' 창에서 아래와 같이 설정해 주세요.
유형: 톱니바퀴(⚙️) 클릭 → '웹 앱' 선택
실행 계정: '나(내 계정)' 선택
액세스 권한: '모든 사용자' 선택

설정 완료 후 오른쪽 아래 '배포' 버튼을 클릭하세요.
5
권한 승인하기 (처음 배포할 때만)
구글 권한 승인 화면이 나타나면 '권한 검토' → 계정 선택을 진행하세요.
'Google에서 확인하지 않은 앱' 경고가 뜰 수 있는데 정상이에요.
⚠ '고급' 클릭 → '(앱 이름)으로 이동(안전하지 않음)' 클릭 → '허용'
kimju.zip 앱은 개인 정보를 수집하지 않으니 안심하세요!
6
웹 앱 URL 복사해서 사용하기
권한 승인이 완료되면 웹 앱 URL이 표시돼요.
옆의 '복사' 버튼(📄)을 눌러 URL을 복사하고 즐겨찾기나 필요한 곳에 붙여넣으세요.

설치 완료! 이제 선생님만의 앱이 준비됐어요 🎉
